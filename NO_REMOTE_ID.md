# Remote ID Policy

DronePeak FCC does not include Remote ID disabling and will not add it.

## Short Version

FCC profile application, CE restore, 4G frame sending, LED control, and device information queries are operational features that can have legitimate authorized uses depending on the user's region and hardware.

Remote ID disabling is different. In many regulated airspaces, Remote ID is a legal safety requirement. Disabling it can create serious regulatory and safety risk.

## Safety Position

Remote ID helps authorities and nearby airspace participants identify drone activity. Removing or disabling that signal can make a drone harder to detect in situations where visibility matters for public safety.

DronePeak FCC is not designed to hide aircraft identity, bypass Remote ID law, or interfere with aviation safety systems.

## Scope

DronePeak FCC will focus on:

- FCC profile application where authorized.
- CE restore.
- Keepalive behavior.
- 4G activation frame delivery for supported enterprise hardware.
- LED control.
- Device information and diagnostics.
- Safe source-level merging from the upstream FreeFCC project.

Remote ID disabling is out of scope.
